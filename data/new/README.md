# Daily New Registrations

New domains registered with NameSilo (IANA #1479), organized by date.

```
new/
└── 2026/
    ├── 06/
    │   ├── 2026-06-20.txt
    │   └── ...
    ├── 2026-06.txt
    └── ...
```

Each `.txt` file contains one domain per line, sorted alphabetically.
Collection starts from the date `NETAPI_TOKEN` was added to GitHub Secrets.
Updated daily at 06:00 UTC.
